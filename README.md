# Anon Chat

## Backend

- Uses `uv` for dependency management.
- Deployed to `fly.io` using the `Dockerfile`.

## Frontend

- Moved under frontend folder as a git submodule.
- Written in Vue and uses PrimeVue and tailwindcss.
- `npm install` and `npm run dev`.
- Deployed to `fly.io` using the `Dockerfile`.
